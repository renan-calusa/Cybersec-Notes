


## 📁 About This Repository

This repository is a curated collection of my personal notes on **Offensive Security** and related cybersecurity topics. It covers a wide range of areas, including:

- Web Security
- Penetration Testing
- Networking
- Reverse Engineering
- Binary Exploitation
- Operating Systems
- And more

The goal of this project is to organize and share structured learning material, references, and practical knowledge gathered throughout my studies and hands-on experience.

## 🚧 Work in Progress

I am currently restructuring and improving all notes to make them clearer, more consistent, and easier to navigate.

Because of this ongoing process, some folders may be incomplete or temporarily empty. Content will be added and refined gradually over time.

## 📌 Recommended Usage

For the best experience, it is recommended to:

1. **Clone the repository**, or download the latest [**release**](https://github.com/renan-calusa/Cybersec-Notes/releases).  
2. Open the notes using [**Obsidian**](https://obsidian.md/).

These features make exploring and studying the notes significantly more effective.
