
> The DOM is a structured representation of a webpage that JavaScript can read and modify.
> 
> A **DOM tree** is a *tree structure* whose nodes represent an HTML or XML document's contents. Each HTML or XML document has a DOM tree representation.

This code would generate this Tree, for example:

```html
<html lang="en">
  <head>
    <title>My Document</title>
  </head>
  <body>
    <h1>Header</h1>
    <p>Paragraph</p>
  </body>
</html>
```
